### 미니체결
미니체결은 바이낸스 USDT 선물의 체결 내역을 원하는 거래대금으로 필터링할 수 있는 앱입니다.

일반적으로 거래소에서 보는 체결내역은 너무나 빠르게 지나가고 작은 단위의 거래도 보여줍니다.

이것은 유저의 피로감을 증가시키는 원인이 되는데요. 

미니체결을 사용하면 천만원 혹은 1억 이상의 큰손들의 체결내역만 볼 수 있습니다.

진짜 큰손들의 거래내역을 손쉽게 팔로우해보세요.
